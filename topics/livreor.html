Un livre d'or

Interaction avec l'utilisateur

Un livre d'or est un module pour une application web permettant à des gens de laisser un message/avis sur l'application. Même si ce n'est plus particulièrement d'actualité, cet exercice vous propose de réaliser un petit livre d'or, soit une page web où des gens peuvent consulter ou laisser des messages. Vous pouvez biensur allez plus loin en laissant la possibilité de laisser une note et éventuellement paginer les résultats.

Pour les systèmes un peu exotique, n'hésitez pas à rédiger un README pour expliquer comment "builder" et "lancer" votre petite application !
