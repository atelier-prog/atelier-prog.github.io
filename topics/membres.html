Un espace membre

Inscription, connexion, profil

Il est très courant pour une application web de laisser les utilisateurs se créer un compte pour interagir avec l'application. Dans cet exercice on vous propose de permettre aux visiteurs de se créer un compte, de le modifier, et de se connecter pour accéder à certaines pages du site web réservées aux membres enregistrés.

Vous pouvez en profiter pour considérer le livre d'or comme une page nécéssitant un compte.

Pour les systèmes un peu exotique, n'hésitez pas à rédiger un README pour expliquer comment "builder" et "lancer" votre petite application !
